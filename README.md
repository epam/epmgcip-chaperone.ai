## Steps to run:
Configure venv from the root:
1. python -m venv .venv  
2. .venv\Scripts\activate (Windows)
2.1. Configure python interpreter path from .venv in VS Code
3. pip.exe install -r .\requirements.txt 
4. pip install -e .
